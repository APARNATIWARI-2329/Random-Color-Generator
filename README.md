# 🎨 Random Color Generator

A modern Random Color Generator built using HTML, CSS, and JavaScript. The application dynamically generates colorful palettes and allows users to copy color codes with a single click.

---

## 🚀 Features

* Generate random color palettes
* Copy color codes to clipboard
* Interactive UI
* Responsive design
* Modern card-based layout

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* DOM Manipulation

---

## 📚 Concepts Practiced

* DOM Manipulation
* Event Handling
* Dynamic Element Creation
* Random Number Generation
* Clipboard API
* Responsive Design

---

## 🎯 Learning Outcomes

This project helped me improve my understanding of:

* JavaScript DOM operations
* Dynamic UI generation
* Event listeners
* Color manipulation
* Building interactive web applications

---

## 💡 How It Works

1. Click **Generate Colors**
2. Six random colors are generated
3. Click any color card
4. Color code is copied to clipboard
